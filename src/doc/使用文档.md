# LtzlReport

联通助理埋点工具

## 引入插件

```html
<script src="./ltzl-report.js"></script>
```

## 初始化 & 调用

```js
// 初始化
const ltzlReport = window.LtzlReport({
  appid: "your appid"
})

// 查看版本信息
console.log(ltzlReport.version)

// 埋点函数
ltzlReport.report({
  event: "event name"
})
```

## PV & UV

插件初始化成功后默认上报页面打开和关闭事件。
